----------------------------
; Ауры монстров
----------------------------

В данный момент реализованы огненная, психическая и радиационная аура
для каждой из аур, существуют параметры:


max_distance		- 	максимальная дистанция на которой действует аура (по умолчанию 0)
max_power			-	максимум влияния (удобно для ограничения на близком растоянии)
linear_factor, quadratic_factor	-	два кофициента формулы влияния:
					aura strength(distance) = linear_factor/distance + quadratic_factor/distance*distance
					distance - растояние до источника (монстра)
					linear_factor - для медленно(линейно) убывающего поля
					quadratic_factor - для быстро убывающего (подобно гравитации) поля

pp_effector_name		    -	пост-процесс эффектор 
pp_highest_at			    -	сила ауры при которой уровень пост-процесса максимальный
sound                       -   звук ауры (громкость будет максимальной при макс. уровне пост-процесса)

пример настроек для конкретных аур:

fire_max_distance			=	10
fire_max_power				=	5
fire_linear_factor			=	0
fire_quadratic_factor		=	0.025

psy_max_distance			=	10
psy_max_power				=	5
psy_linear_factor			=	0.05
psy_quadratic_factor		=	0

radiation_max_distance		=	10
radiation_max_power			=	5
radiation_linear_factor		=	1
radiation_quadratic_factor	=	1

----------------------------
; Невидимость Бладсакера
----------------------------
настраивается параметрами

full_visibility_radius - радиус меньше которого кровосос полностью видимый (5 по умолчанию)
partial_visibility_radius - радиус меньше которого кровосос частично видимый (10 по умолчанию)
на растоянии большем чем partial_visibility_radius кровосос невидимый

visibility_state_change_min_delay - таймаут меньше которого не может быть между состояниями видимости (1000 мс по умол)

----------------------------
; детекция полтергейста
----------------------------

параметры для настройки детекта телекинетических полтергейстов

detection_post_process		=	effector_nightvision_good   (название пост-процесс-эффекта)

зп: detection_pp_effector_name  =	poltergeist_detection_effector  (название пост-процесс-эффекта)

два параметра для установки зависимости силы детекта от растояния до актёра
сила детекта считается по формуле detection_linear_factor/distance + detection_quadratic_factor/(distance*distance)

detection_linear_factor		=	10.0
detection_quadratic_factor	=	0.0

скорость уменьшения текущего уровня детекта
detection_loose_speed		=	5.0

максимальный уровень на котором действует детект
detection_max_distance		=	20.0

уровень детекта на котором полтергейст может кидать физические предметы
detection_success_level		=	4.0
